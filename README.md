# React Starter


## Creation Process

### Create **[React](https://react.dev)** app template with **[Vite](https://vitejs.dev)** build tool

```
npm create vite@latest react-starter  -- --template react
```

### Goto **project** folder

```
cd react-starter
```

### Install tailwindcss, postcss, autoprefixer and daisyui

```
npm i -D tailwindcss postcss autoprefixer daisyui@latest
```

### Generate **tailwind** and **postcss** config files

```
npx tailwindcss init -p
```

### Run dev environment

```
npm run dev
```
